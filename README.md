
# 📚rainforest
Rainforest, a library of Java utility classes for Java 8+

## 🛠️Environment（开发环境）
+ JDK 8
+ Apache maven 3.6.1


## 💿集成方式
### Maven
```xml
<dependency>
  <groupId>com.iofairy</groupId>
  <artifactId>rainforest</artifactId>
  <version>0.2.0</version>
</dependency>
```

### Gradle
```
implementation 'com.iofairy:rainforest:0.2.0'
```
